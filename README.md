Это небольшое веб-приложение на Streamlit, для редактирования изображения с использованием генеративной модели


В качестве генеративного движка я выбрала Stability AI API (модель Stable Diffusion 3) в режиме Image-to-Image. Пробовала сделать через OpenAI Image Editing API и Replicate, но они все просят денежек для работы


**Основной функционал:**
Загрузка изображений (JPG, PNG), далее отправка фото и промпта в API генеративной модели и отображение результата в формате До/После». Есть выбор заготовленных промптов (про курение и алкоголь), возможность скачать результаты. Также я немного постаралась над эстетикой и интерфейсом) 

## Как запустить проект локально
1. **Склонируйте репозиторий:**
   ```bash
   git clone <ссылка-на-ваш-репозиторий>
   cd <название-папки>
2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   python -m venv venv
   venv\Scripts\activate
3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
4. **Настройте переменные окружения:**

    Зарегистрируйтесь на platform.stability.ai и получите бесплатный API-ключ и вставьте его в .env
    ```bash
    STAB_API_KEY=...
5. **Запустите приложение**
   ```bash
   streamlit run app.py
